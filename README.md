## 스케줄 관리 캘린더

### 기능

1. 월별 스케줄을 입력  
   주간,야간,비번,휴무를 앞글자만 따서 그 월의 일수대로 작성한다  
   입력 형식: 주주주야비휴...

2. 각 일마다 기상 시간을 입력  
    기상시간을 4:30, 6:00 중에 선택할 수 있다.  
   내가 해당 비번일에 일어난 기상시간을 메모하는 용도

3. 함께 근무한 사람을 입력  
   함께 근무한 사람을 기록한다.

### 저장방법

브라우저의 localstorage에 저장한다.
다른 브라우저 키면 저장안되어 있음 유의

### 사용 기술스택

v0 사용하여 개발  
ts + vite + react + tailwind

### 배포

GitHub Pages를 통해 자동 배포됩니다.

1. GitHub 리포지토리에서 Settings > Pages로 이동
2. Source를 "GitHub Actions"로 설정
3. `main` 브랜치에 푸시하면 자동으로 빌드 및 배포됩니다

**참고:** 리포지토리 이름이 다르다면 `vite.config.ts`의 `repoName` 변수를 수정하세요.
